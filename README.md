# Reestar - Food Ordering Website

![Reestar Screenshot](/screenshot.png)

A simple food ordering website built with HTML and basic CSS.

## Features

- Homepage with food items
- Order buttons for each item
- Contact form section
- Responsive design

## Technologies Used

- HTML5
- CSS3
